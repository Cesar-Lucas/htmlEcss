Principais formatações

Destaques / negrito

nesta frase temos um termo em negrito usando a tag "B"(não semantica)

Nesta frase, temos um termo em destaque usando a tag "strong"(semantica)

Itálico / Ênfase

agora temos uma frase em destaque com italico usando a tag "i"(não semantico)

Nesta frase, temos um termo em ênfase usando a tag EM(semantico)

Textos marcados

Podemos criar tambem um texto marcado usando a tag "MARK"

Usando CSS podemos deixar todos os marques da cor que querermos.

Texto Grande e Texto Pequeno

Estamos criando um texto grande(a tag big esta obsoleta, use CSS para deixar letras grandes) e outro pequeno(letra pequena continua em uso, mas deve ser usada com significado, se quer deixar a letra menor use CSS) dentro deste paragrafo.

Texto Excluido

podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado

Texto Inserido

podemos maracar um texto como inserido para dar uma êngase e indicar que ele foi adicionado depois

Texto Sobrescrito

para inserir coisas tipo: x20+3

Texto Subscrito

para inserir coisas tipo : h2o
